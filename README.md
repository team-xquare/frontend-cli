## Xnew?

XQUARE-frontend 모노레포에서 서비스 혹은 라이브러리를 추가하는 것을 도와주는 cli입니다.
xquare-frontend 레포지토리에서 서비스 혹은 라이브러리를 추가를 도와주기 위해 제작되었습니다.

## 설치 방법

```
npm install -g @xquare/xnew
```

## 사용 방법

### ```xnew -S``` or ```xnew --service```

service를 추가합니다.



------------------------------------------------

### ```xnew -L``` or ```xnew --library```

library를 추가합니다.
